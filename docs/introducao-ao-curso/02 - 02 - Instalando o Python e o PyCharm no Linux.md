Você pode instalar o Python diretamente dentro do PyCharm, de forma semelhante ao Windows, sem a necessidade de instalá-lo previamente pelo terminal. O PyCharm se encarrega de baixar a versão do Python e configurar o ambiente de desenvolvimento automaticamente.

### **Passo 1: Instalar o PyCharm**

1. **Baixar e Instalar:** Acesse o site oficial da JetBrains e baixe a **versão Community** para Linux.  
2. **Extração:** Extraia o arquivo `.tar.gz` que você baixou.  
3. **Execução:** Abra o terminal, navegue até a pasta extraída e execute o seguinte comando para iniciar o software:  
   * `./pycharm.sh`  
4. **Atalho (Opcional):** Você pode criar um atalho ou adicionar o PyCharm ao menu do sistema para facilitar o acesso futuro.

### **Passo 2: Configurar o Projeto e o Python**

1. **Novo Projeto:** Na tela de boas-vindas do PyCharm, clique em **"New Project"**.  
2. **Localização:** Escolha a pasta onde seu projeto ficará em "Location".  
3. **Interpretador Python:** Na seção "Project venv", selecione a versão do Python que deseja instalar (recomenda-se a mais atual). O PyCharm instalará o Python diretamente no projeto.  
4. **Finalização:** Clique em **"Create"** para concluir a criação com o ambiente já configurado.
